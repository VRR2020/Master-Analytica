# Master Analytica Ltda - Site Institucional

Este repositório contém o código-fonte do site institucional da **Master Analytica Ltda**, desenvolvido com o Builder.io e publicado via GitHub Pages em [https://analytica.tec.br](https://analytica.tec.br).

## Sobre a empresa
A Master Analytica é uma consultoria especializada em **Business Intelligence** e **Inteligência de Negócios**, liderada pelo CEO Victor.  
Nosso objetivo é transformar dados em insights estratégicos para apoiar empresas na tomada de decisão e na criação de valor.

## Características do site
- **Design institucional** moderno e responsivo.
- **Destaque para competências** do CEO e da equipe, apresentando experiência e áreas de atuação.
- **Formulário de contato** integrado, que envia leads diretamente para o e-mail: `Victor.have@gmail.com`.
- **Banco de dados de acessos**: registra visitas, localização e consentimento de cookies, criando uma base para futuras estratégias de marketing.
- **Identidade visual**: logo da empresa é uma **coruja preta**, símbolo da pedagogia e da educação, reforçando o compromisso com conhecimento e inteligência.

## Objetivo
O site foi construído com foco em **atrair novos clientes** para serviços de consultoria em Business Intelligence, oferecendo uma experiência clara, confiável e profissional.

## Tecnologias utilizadas
- [Builder.io](https://www.builder.io/) para criação e exportação do layout.
- [GitHub Pages](https://pages.github.com/) para hospedagem gratuita e integração com o repositório.
- HTML, CSS e JavaScript para estrutura e interatividade.
- Integração com banco de dados para registro de acessos e leads.

## Como contribuir
Este repositório é voltado para o desenvolvimento e manutenção do site institucional.  
Sugestões de melhorias podem ser feitas via **Issues** ou **Pull Requests**.
